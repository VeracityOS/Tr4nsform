05 · Builder Output Spec
What the Builder Must Return
Minimal change required. Update output to return Protocol, Pathway, and Tier — not just a peptide list. The peptide list remains as the component layer below.
Step 01
Goals Input
User selects primary goals from the existing 8-option list (fat loss, cognitive, sleep, immunity, performance, anti-aging, mood, recovery)
Step 02
Matching
Goals map to Protocol internally. Builder selects the Protocol and best-fit Pathway based on goal combination and any secondary conditions
Step 03
Tier Assignment
Tier is assigned based on goal complexity, number of goals selected, and optional data source inputs (WHOOP, Oura, bloodwork)
Step 04
Output
Present Protocol → Pathway → Tier as the primary result. Show peptide component stack below as the supporting layer — not the headline

Required Builder output format:

Protocol:    Metabolic
Pathway:     Recomposition Stack
Tier:        Full
Components: Semaglutide · Retatrutide · MOTS-c · 5-Amino-1MQ · CJC-1295 / Ipamorelin
Positioning Rule Never present products as standalone peptide lists. Always present as: Protocol → Pathway → Bundle. TR4RX is the optional component layer — the product is the pathway bundle.
06 · Pricing Structure
Standardised Tier Pricing
Apply consistent pricing logic per tier. Remove any products priced outside these ranges without explicit approval. Apex pricing is retreat-level only.
Entry
$80–100
USDT · per 30-day supply
Low-friction first purchase. Single ingredient or light botanical stack. 30-day supply. Conversion-focused price point. No clinical review required.
Core
$140–195
USDT · per 30-day protocol
Primary offer. 2–3 peptides plus botanicals. 30-day protocol. Main conversion tier. Light intake form. Practitioner note issued.
Full
$240–370
USDT · per 60–90 day bundle
High-value bundle. 3–5 peptides + full stack. 60–90 day protocol. Includes intake review, check-in, and follow-up. Main revenue tier.
Apex
$630–1,000
USDT · retreat-level only
Retreat-level or UHNW only. Full peptide stack + retreat integration + clinical supervision. Executive Stack and Longevity Stack only. Requires consultation.
Pricing Discipline Do not price Entry products as Core. Do not price Apex products without consultation gating. Any product priced outside its tier range must be escalated for approval before going live.
07 · Naming Rules
Product Naming Convention
All CMS products follow [Protocol] — [Pathway] as the primary system name, with the evocative name retained as a subtitle or descriptor. Both layers are required — the system name for structure, the evocative name for brand identity.
❌ Remove These Patterns
"Night Architecture Deep Sleep Program"
"Cognitive Dominance Treatment"
"Anti-Aging Stack Bundle"
"Gut Health Protocol Package"
"30-Day Fat Loss Program"
"Performance Peptide Treatment"
Any name with "program", "treatment", "package"
Any name using the old 12 protocol names as product names
✓ Use This Format
Sleep — Deep Sleep Recovery · for Night Architecture
Brain — Cognitive Enhancement · for Cognitive Dominance

Formula: [Protocol] — [Pathway] · for [Evocative Name] Protocol = the parent category (locked list of 12). Pathway = the specific bundle (locked list of 21). Evocative Name = the brand expression layer — kept as a subtitle or supporting line, never as the primary product name. The evocative name gives the product its identity and aspiration; the protocol/pathway name gives it structure and searchability. Both are required.
